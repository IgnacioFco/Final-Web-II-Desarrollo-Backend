# API de Películas
**Examen Parcial II – Desarrollo Backend (Node.js + Express + MongoDB)**  


## Descripción  

Este proyecto implementa una **API RESTful** que permite gestionar una colección de **películas de terror**.  
Fue desarrollada con **Node.js**, **Express** y **MongoDB Atlas**, utilizando **Mongoose** como capa de modelado de datos.  

El sistema permite:
- **Registrar** nuevas películas con su título, director, año, duración, país, puntuación y sinopsis.  
- **Listar** todas las películas almacenadas en la base de datos.  
- **Buscar** una película específica por su ID.  
- **Actualizar** los datos de una película existente.  
- **Eliminar** una película de la colección.  

El propósito del proyecto es demostrar el uso de un **backend funcional con conexión a una base de datos NoSQL**, aplicando buenas prácticas de desarrollo y modularización de código.


## Tecnologías  
- Node.js  
- Express.js  
- Mongoose  
- MongoDB Atlas  
- Dotenv  
- Nodemon
